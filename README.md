# TechSupport Frontend

## Telepítési út mutató

Miután letölti a zip fájlt és kicsomagolta, Visual Studio Code segítségével nyissa meg, valamint nyisson egy terminált is ott a Ctrl + ö segítségével, majd futassa az alábbi kódot:

```bash
$ npm install
```

Ez után indítsa el az adat bázis host-olását, majd ez után indítsa el a backend-et, ha ezek megvannak futassa az alábbi kódot hogy elinduljon a frontend host-olás:

```bash
$ npm run dev
```
Ez lenne a rendszer felállítás a frontend-hez.

A frontend funkcióinak a leírása és képi megjelenítése az alábbi [linken](https://github.com/Atmerium/TechSupport_Frontend/wiki) érheted el.
